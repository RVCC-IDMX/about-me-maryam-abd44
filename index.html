Animations

What is CSS Animations? CSS Animation is animations you could using CSS rather than using JavaScript. CSS animation isn't just one property, but instead it has a lot of properties that could be used to make websites look cooler. Some of the properties are @keyframes, animation-name, animation-duration, animation-delay, and many more. @keyframes are inserted in a lot of different CSS, which means it can be reused. Next is animation-name, this property is used with @keyframes, it gives a specific name for it. The next property is animation-duration, is when telling the code, a specific time when the animation needs to be stopped. Furthermore, animation-delay is used when you want to tell the code to start the certain animation later. There are so many things you could of with CSS animation that someone can create.

I decided to make an animated circle that moves across the screen. At first, it was hard trying to understand how all these properties work. With many mistakes made I made, I didn’t stop even though it got frustrating. When I started to understand how they work, it got really fun because there is so much that someone could do.
